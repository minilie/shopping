# 项目申请书（精简版）

## 项目名称
基于 HarmonyOS ArkTS 的商城商品列表应用。

## 背景与目标
面向移动互联网编程实践课程，目标是完成一个可滚动、支持下拉刷新与懒加载的商品列表页面，并形成完整的需求、架构与测试材料。

## 功能亮点（当前版本）
- 多分类 Tab 商品列表（精选/手机/服饰/穿搭/家居）。
- 下拉刷新与滚动懒加载。
- 触底提示与空状态提示。
- 搜索、价格区间筛选与排序。

## 技术路线
- 语言与框架：ArkTS + HarmonyOS UI 组件（`List`、`Tabs`、`Scroll`、`LazyForEach`）。
- 数据组织：本地模拟数据源 `ProductListDataSource`，分页加载与刷新。
- 结构分层：`pages/`（页面）、`view/`（组件）、`viewmodel/`（数据）。

## 计划与风险
- 计划补齐真实网络接口接入与真机性能测试。
- 风险在于接口联调与性能指标量化，需预留测试时间。
